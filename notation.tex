\section{Notes on Notation}

\begin{enumerate}
\item \textbf{Coordinate Points}

For any two points $X = (x_1, x_2, \hdots, x_n)$ and $Y = (y_1, y_2, \hdots,
y_n)$, we say $X < Y$ if and only if $x_i < y_i$ for all $1 \leq i \leq
n$. The relationships of $\leq, >, \geq,$ and $=$ are all defined similarly. 
\end{enumerate}
